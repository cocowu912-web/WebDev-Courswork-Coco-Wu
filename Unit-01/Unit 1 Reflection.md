# Unit 1 Reflection

[Reflect on your journey here]
In this unit, I learned the basic structure of HTML and how to format text on a web page. At first, I thought HTML was just about typing words, but now I understand that every web page needs a proper structure with <html>, <head>, and <body> tags.
The most useful thing I learned was the difference between visual tags and semantic tags. For example, <b> and <strong> look the same, but <strong> tells screen readers that the text is important. This made me realize that writing good HTML is not only about how the page looks, but also about making it accessible to everyone.
I also practiced using various text formatting tags like <i>, <em>, <del>, <ins>, <mark>, and <small>. The code challenge asked me to create a simple web page using at least 6 of these tags, which helped me understand when to use each one appropriately.
One thing I found a bit confusing was remembering to close tags in the correct order when nesting them. I sometimes forgot that the inner tag must be closed before the outer tag. However, after a few tries, I got used to it.
