# Unit 11 Reflection

[Reflect on your journey here]
When I started this course, I felt nervous because I had never coded before. I did not know if I could learn HTML, CSS, and JavaScript. I was worried it would be too difficult. But now I feel proud because I can create a real website.
The most important thing I learned was the Design Thinking Process. Before this course, I thought design was just about making things look nice. Now I know that good design starts with understanding the user's needs. Interviewing my client Kai helped me think about what they really want, not just what I like.
I also learned that HTML is for structure, CSS is for style, and JavaScript is for interactivity. At first, I mixed them up, but now I understand their different jobs. My favorite part was CSS because I like choosing colors and making the page look good. The hardest part was JavaScript because functions and getElementById were confusing at first.
Accessibility was a new idea for me. I did not know that some people use# screen readers to visit websites. This made me understand why semantic tags like <strong> and <em> are better than <b> and <i>. I also learned why the alt attribute is important for images.
For my community website, I chose Goal 6: Clean Water. I created a three-page website with a navigation bar, buttons, and responsive design. I am proud of this project because it can help real people learn about water problems.
In the future, I want to learn more advanced JavaScript and maybe try React. I am also interested in UX design because I like thinking about how users feel. I hope to build a portfolio and maybe do freelance work one day.
This course gave me a solid foundation in web design. I am excited to keep learning and improving my skills.
