# Unit 8 Reflection

[Reflect on your journey here]
In this unit, I learned advanced CSS techniques to improve my website layout. Before this, my pages were simple with basic styles. Now I can use <div> and class to organize content into sections.
The most useful thing I learned was the box model. I now understand that every element has content, padding, border, and margin. This helps me control spacing and layout better. I used padding inside my header and margin around my content boxes.
I also learned about CSS positioning. I used static for normal flow and relative for small adjustments. For my action page, I used display: inline-block to put two boxes side by side. This makes the page look more organized.
Responsive design was interesting but a bit difficult. I learned about fluid grids using percentages, flexible images with max-width: 100%, and media queries to change styles at different screen sizes. I created a responsive page with three columns that stack on small screens.
For adaptive design, I used breakpoints at 768px and 480px to change colors and font sizes for tablets and phones. I also learned about the viewport meta tag which tells the browser how to scale the page.
One challenge was choosing between responsive and adaptive design. Responsive is more flexible but adaptive gives more control. I think responsive is better for most websites because it works on all screen sizes.
Overall, this unit made my website look more professional.
