# Unit 10 Reflection

[Reflect on your journey here]
In this unit, I learned how to organize my website files and create a navigation bar. Before this, my files were all in one place and my navigation was just simple links. Now I have a proper folder structure with an images folder.
The most useful thing I learned was the <nav> element and how to make a horizontal navigation bar. I used <ul> and <li> with list-style-type: none to remove bullets, and float: left to make links horizontal. I also added overflow: hidden to keep the layout clean. For mobile screens, I used a media query to make the navigation vertical.
I also learned about the five design principles: visual hierarchy, consistency, navigability, color theory, and typography. I applied visual hierarchy by making my h1 large and centered. I kept consistency by using the same navigation and colors on all three pages. I chose blue colors because they match the water theme.
One thing I found difficult was understanding float: left at first. I did not know why the navigation background disappeared, but then I learned that overflow: hidden fixes this problem.
Overall, this unit made my website more professional and easier to use.
