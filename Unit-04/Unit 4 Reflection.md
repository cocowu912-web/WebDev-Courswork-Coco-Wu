# Unit 4 Reflection

[Reflect on your journey here]
In this unit, I learned about HTML attributes and how to make my website more accessible. The most useful new thing was the lang attribute because it helps screen readers know what language the page is in.
I also learned how to add lists to my web page. Before this, I only used paragraphs. Now I can use ordered lists for steps, unordered lists for random items, and description lists for terms and definitions. I added a nested list to show my favorite natural materials and their types.
For images, I learned that the alt attribute is very important for people who cannot see the image. It also shows text when the image fails to load. I added Kai's photo with width and height attributes.
Accessibility was a new idea for me. I did not know that some people use screen readers or braille displays to visit websites. This made me understand why semantic tags like <strong> and <em> are better than <b> and <i>.
One thing I found a bit confusing was the difference between absolute and relative URLs for images. But I think using a relative URL is easier when the image is in the same folder.
Overall, this unit helped me improve Kai's web page with more structure and accessibility.
